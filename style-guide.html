Brand Usage Guidelines

Follow these rules to maintain brand consistency.

Hot Pink Usage

DO

  • Use for primary CTA buttons
  • Use for "BANK ON FRANK!" tagline
  • Use for section subtitle labels
  • Keep usage to 10% of page

DON'T

  • Use for body text or paragraphs
  • Use for headlines (use Dark Azure)
  • Overuse throughout the page
  • Use for backgrounds

Gold Usage

DO

  • Use for stats and numbers ($1B+, 30 Years)
  • Use for phone numbers in CTAs
  • Use for achievement highlights
  • Use for icons on dark backgrounds

DON'T

  • Use for body text
  • Use for primary buttons
  • Use on light backgrounds without contrast
  • Overuse outside stats/achievements

Typography

DO

  • Use Montserrat for all headings
  • Use Arial for body text
  • Use Extra-Bold (800) for taglines
  • Maintain consistent heading hierarchy

DON'T

  • Mix other fonts into the design
  • Use Montserrat for long paragraphs
  • Skip heading levels (H1 to H3)
  • Use light font weights for headlines

CSS Variables Reference

Use these CSS custom properties in your stylesheets.

/* Brand Colors */
--charcoal: #2C2C2C;       /* Body text, footer */
--dark-azure: #033563;     /* Headlines, nav, links */
--pastel-azure: #A2DCF5;   /* Backgrounds, icons */
--gold: #FFD700;           /* Stats, achievements */
--hot-pink: #E00D93;       /* CTAs, taglines ONLY */

/* Typography */
--font-primary: 'Montserrat', sans-serif;   /* Headlines, buttons */
--font-secondary: Arial, Helvetica, sans-serif;  /* Body text */

/* Spacing */
--spacing-sm: 0.5rem;
--spacing-md: 1rem;
--spacing-lg: 1.5rem;

--spacing-2xl: 3rem;
--spacing-3xl: 4rem;

/* Border Radius */

/* Transitions */
